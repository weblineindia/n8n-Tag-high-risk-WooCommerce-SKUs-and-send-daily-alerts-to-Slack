# n8n-Tag-high-risk-WooCommerce-SKUs-and-send-daily-alerts-to-Slack
Stay ahead of inventory risks with this **n8n workflow automation**. It analyzes WooCommerce sales over the last 14 days, automatically tags fast-selling SKUs by risk level, and sends actionable Slack alerts. A practical **n8n workflow** for inventory monitoring, stock planning, and eCommerce operations.
